# ACV Advanced Data Analytics Training

This repository contains the case studies completed during the **Ahmet Çalık Vakfı Advanced Data Analytics Training** program.

The projects focus on applying data analysis techniques to real-world inspired datasets from different domains such as energy consumption and financial customer analytics.

⚠️ Note:  
These projects were developed as part of a training program. Some parts may be incomplete or simplified, but they demonstrate the analytical workflow including:

- Data exploration
- Data cleaning
- Visualization
- Dashboard design
- Business insight generation
